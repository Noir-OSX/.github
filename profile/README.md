## Overview

<a href="#">Noir</a> brings dark mode to every website in Safari, regardless of whether sites support dark mode natively. The extension analyzes page content and applies intelligent styling that preserves readability while reducing eye strain from bright white backgrounds. Comfortable browsing extends to the entire web.

The <a href="#">automatic dark mode</a> activates without manual intervention. New websites receive dark styling automatically. No need to configure each site individually. Automatic operation makes dark browsing effortless.

<a href="#">Intelligent styling</a> produces readable dark pages. Algorithms adjust colors for proper contrast. Text remains readable against dark backgrounds. Styling intelligence avoids broken appearances.

The <a href="#">image handling</a> preserves visual content appropriately. Photos display normally rather than inverted. Diagrams and graphics maintain clarity. Image intelligence prevents visual distortion.

<a href="#">Per-site customization</a> refines dark mode for specific sites. Adjust darkness levels for individual websites. Disable dark mode where not wanted. Customization addresses sites where automatic styling needs refinement.

The <a href="#">brightness control</a> adjusts overall page dimness. Make pages darker or lighter as preferred. Brightness adjustment complements dark mode. Control enables personal comfort tuning.

<a href="#">Schedule activation</a> enables dark mode at appropriate times. Activate at sunset automatically. Follow system dark mode settings. Scheduling aligns dark browsing with environmental lighting.

The <a href="#">Safari integration</a> feels native and performs well. Extension operates efficiently within Safari. No browser performance impact. Integration quality matches Apple's standards.

<a href="#">Privacy focused</a> operation keeps browsing data local. No website content sent externally. Local processing protects privacy. Privacy architecture respects user data.
